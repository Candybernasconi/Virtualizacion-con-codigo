# Virtualizacion-con-codigo
* *Virtualización:* Entorno gestionado en VirtualBox utilizando controladores de red en modo Adaptador Puente para integración en red local.
* *Sistema Operativo:* Ubuntu Server (gestión vía terminal).
* *Gestión de Código:* Repositorio en GitHub configurado para la sincronización y ejecución del script código.py.
* *Conectividad:* Acceso remoto habilitado para dispositivos externos dentro del mismo segmento de red local.
